AI助手对话示例-开发中

使用说明

  • 点击右下角的聊天图标打开对话窗口
  • 点击"华微世纪AI助手",可以打开操作指南页面.
  • 打开对话窗口时，再次点击右下角的聊天图标关闭对话窗口
  • 拖拽对话窗口左上角，调整窗口大小
  • 点击“x”按钮关闭对话窗口
  • 点击“+”按钮最大化对话窗口
  • 点击“-”按钮最小化对话窗口
  • 点击“#”按钮显示/隐藏工具栏
